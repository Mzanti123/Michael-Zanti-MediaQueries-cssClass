Inline-Block lets you set set the height and width of an element, but at the same time respecting the top and bottom margins and padding of that element.

Display:Block means that the element is displayed in a block. The block will have some whitespace above and below it. Display:Block will not tolerates no HTML elements next to it unless ordered by additional elements.

This block does not have any text alignments.
